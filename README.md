# Notes App 📝

A simple and responsive Notes Application built using Flutter and Dart for creating, editing, saving, and deleting notes.

---

## Features

- Add Notes
- Edit Notes
- Delete Notes
- Save Notes
- Responsive UI
- Simple and Clean Design
- Beginner Friendly

---

## Technologies Used

- Flutter
- Dart

## How to Run

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/notes-app.git
```

---

### 2. Open Project Folder

```bash
cd notes-app
```

---

### 3. Install Dependencies

```bash
flutter pub get
```

---

### 4. Run Application

```bash
flutter run
```
